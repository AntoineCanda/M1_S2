Pour lancer le projet :

mvn clean package tomee:run

Pour utiliser le serveur :

Se connecter sur la page d'init qui se trouve en 
	http://localhost:8080/books
	
Pour ajouter un livre :
	Cliquer sur ajouter un livre, remplir les champs nécessaires et valider
	
Pour rechercher un livre :
	Commencer par sélectionner le type de recherche (sur le titre ou l'auteur)
	Remplir le champs associé recherche
	Valider
	
Pour trier selon les années de parution :
	Appuyer sur Year dans la liste des livres 
	
Pour regarder le detail d'un livre : 
	Cliquer sur nom du livre

Pour regarder la commande en cours :
	Cliquer sur Panier
	
Pour regarder toute les paniers :
	Cliquer sur Les paniers

Pour ajouter un livre au panier :
	Cliquer sur ajouter au panier
